**Описание проекта**

Страница для изучения HTML, CSS, адаптивной верстки.

**Технологии и инструменты**

* флексбокс-верстка;
* grid layout; 
* позиционирование элементов;
* создана файловая структура по БЭМ (Nested);
* реализована адаптивная верстка; 
* часть логики написана на JS.
